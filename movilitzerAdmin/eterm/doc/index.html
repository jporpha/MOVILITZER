A. Top Header is defined in CSS as - - - - - ".header".
B. Sub Menu is defined in CSS as - - - - - ".tabs-contenor".
C. Main is defined in CSS as - - - - - "#main". this div contains two divs "#right-menu" and "#content".

B) CSS Files and Structure - top

I'm using nine CSS files in this theme.

1). reset.css
2). style.css
3). custom.css
4). data-table.css
5). elfinder.css
6). fullcalender.css
7). ui-slider.css
8). jquery.fancybox.css
9). wysiwyg.css

The first one ( reset.css ) is a generic reset file. Many browsers interpret the default behavior of html elements differently. By using a general reset CSS file, we can work round this. This file also contains some general styling, such as anchor tag colors, font-sizes, custom fonts etc. Keep in mind, that these values might be overridden somewhere else in the file.

The second file ( style.css ) contains all of the specific stylings for the page. The file is separated into sections using comments like this

/*===============TOP HEADER AND MENU==================*/
/*===============TOP HEADER AND MENU==================*/


          all stles for top header and menu
					   
					   
/*===============RIGHT MENU START==================*/
/*===============RIGHT MENU START==================*/


             all stles for right menu

Rest of the css is used as per there naming conventions like data-table.css contains grid styles, elfinder.css contains file manager styles, fullcalendar.css contains calendar styles, etc.

C) JavaScript - top

This theme imports many Javascript files.

  1. custom.js - Most of the functionalities in this theme is carried out from the custom scripts.
  2. jquery.js - This is an jquery library.
  3. jquery.min.js - This is an min jquery library.
  4. elfinder.full.js, elfinder.full.js - File maneger is useing this scrip, file maneger is in plugin page.
  5. FilterScript.js - This script is for text filtering on forms elements page.
  6. js/datepicker - All scripts in this folder are used for date picker.
  7. js/charts - All scripts in this folder are used for Charts.
  8. forms-style.js - To change the look and feel of radio and checkboxes i have used this script.
  9. formwizard.js - Step by Step is maneged from this script.
  10. fullcalendar.min.js - This is used for calendar.
  11. jquery.dataTables.js - To make dynamic gride (table).
  12. jquery.miniColors.js - For colorpicker.
  13. StickyScroller.min.js, vertlib.min.js - To make fixed the position of right menu after some scroll.
  14. slidernav.js - Contact List is maneged from this script.
  15. maskedinput.js - For making the input filds masked with some value in forms page.
  16. jquery.tipsy.js - For Tooltips.
  17. jquery.simplemodal.js - Pop up Modalbox in notification page.
  18. jquery.purr.js - Sticky notes.
  19. jquery-barcode-2.0.2.min - Barcode Generator.
  1. jQuery is a Javascript library that greatly reduces the amount of code that you must write.
  2. Most of the animation in this site is carried out from the customs scripts. There are a few functions worth looking over.
  3. In addition to the custom scripts, I implement a few "tried and true" plugins to create the effects. This plugin is packed, so you won't need to manually edit anything in the file. The only necessary thing to know is how to call the method.
